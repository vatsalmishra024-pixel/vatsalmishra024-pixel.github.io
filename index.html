Research Overview

Developed an advanced vision-based driver drowsiness detection system under the supervision of Prof. Monideepa Das at Illinois Institute of Technology. The system employs sophisticated fuzzy logic algorithms combined with real-time facial feature recognition to detect early signs of driver fatigue, potentially preventing accidents caused by drowsy driving.

Technical Implementation

  • Implemented real-time facial landmark detection using OpenCV and Dlib for tracking eye aspect ratio (EAR) and mouth aspect ratio (MAR)
  • Designed fuzzy logic inference system with membership functions for drowsiness levels (Alert, Drowsy, Severely Drowsy)
  • Developed Python-based real-time video processing pipeline achieving 30+ FPS on standard hardware
  • Applied histogram equalization and adaptive filtering for robust performance in varying lighting conditions
  • Integrated audio-visual alerting mechanism with graduated warning levels based on drowsiness severity
  • Implemented temporal smoothing algorithms to reduce false positives from legitimate eye blinks
